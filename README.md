# frontend-meetups-in-NL
A list of meetups and initiatives for front-end developers in the Netherlands! Feel free to fork and open a pull-request with additions or removals of inactive meetups.

Below is the general list. I also made specific lists for 
- [meetups themed around a certain CMS](meetups-cms.md)
- [meetups aimed at women/increasing diversity in tech](meetups-for-womenintech.md)
- [initiatives where you can learn to code (front-end)](learntocode.md)
